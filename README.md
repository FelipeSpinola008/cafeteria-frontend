# ☕ CoffeeDev - Landing Page

Esta é a primeira etapa de um projeto Fullstack de uma cafeteria artesanal. O foco desta fase foi construir uma interface premium, responsiva e com navegação fluida.

## 🚀 Tecnologias Utilizadas
* **React.js** (Vite)
* **Tailwind CSS** para estilização utilitária
* **Lucide React** para iconografia
* **JavaScript** (ES6+)

## 🛠️ Funcionalidades Implementadas
- **Carrossel de Produtos Personalizado**: Desenvolvimento de um slider horizontal utilizando `useRef` e `scrollIntoView` para navegação suave, sem dependências externas.
- **Navegação Inteligente**: Implementação de ancoragem por IDs no Header e Footer para scroll suave entre seções.
- **UI Customizada**: Estilização da barra de rolagem (scrollbar) para harmonia com a paleta de cores do projeto.
- **Componentização**: Estrutura organizada para facilitar a futura integração com API e Banco de Dados.
